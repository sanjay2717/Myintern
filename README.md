## What is Myintern?

Myintern is a free, lightweight AI-based internship 
recommendation engine built for the PM Internship Scheme. 
It takes 4 simple inputs from a candidate — education, 
skills, sector interest, and location — and returns the 
top 5 most relevant internships using TF-IDF cosine 
similarity matching. Results are shown as clean visual 
cards with a match percentage score.

---

## Tech Stack

- Python Flask — Backend server
- scikit-learn — TF-IDF recommendation engine
- Cosine Similarity — Match scoring
- HTML CSS JavaScript — Frontend (no frameworks)
- JSON — Internship data storage
- Render.com — Free cloud hosting

---

## How to Run Locally

Step 1 — Install dependencies
pip install -r requirements.txt

Step 2 — Start the server
cd backend
python app.py

Step 3 — Open in browser
http://localhost:5000
